# Unknown Word Handling in POS Tagging

This project implements a Character-aware BiLSTM POS Tagger
to handle Out-of-Vocabulary (OOV) words using Universal Dependencies.

## Features
- Character-aware BiLSTM
- OOV handling
- Universal Dependencies dataset
- Interactive testing

## Model
Character-aware BiLSTM POS Tagger
